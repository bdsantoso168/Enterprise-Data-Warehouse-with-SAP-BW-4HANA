# Enterprise Data Warehouse with SAP BW/4HANA
**ISOM 230 | Spring 2026 | Benedict Daxell Santoso**

## What this is
A term project building a full business intelligence pipeline for 
Global Bike Inc. - a multinational cycling company - using SAP 
BW/4HANA 2.0 and Eclipse Modeling Tools.

This covers the complete BI lifecycle: data ingestion, dimensional 
modeling, ETL pipeline design, master data management, and analytics.

## What was built
- Designed DataSources for 171,012+ raw sales records across Germany 
  and US with different locale formats
- Modeled InfoObjects (key figures and characteristics) as reusable, 
  company-wide data objects
- Built full ETL data flows with transformation rules including formula 
  calculations, master data lookups, and constant value rules
- Loaded master data (29 materials) and transactional data into an 
  advanced DataStore Object (aDSO)
- Project ongoing - dashboards and predictive analytics coming in BM6-8

## Tools used
SAP BW/4HANA 2.0, Eclipse Modeling Tools, SAP Fiori, SAP GUI
